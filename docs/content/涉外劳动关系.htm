第二十章 涉外劳动关系

涉外劳动合同，是指劳动合同中存在境外因素。境外指除中国大陆地区以外的国家和地区，包括香港特别行政区、澳门特别行政区、台湾地区、外国国家和地区、公海和南极大陆。

涉外劳动合同可以表现为：

  1. 劳动者是外国人、无国籍人或者港澳台居民。
  2. 劳动者是中国公民，但是经常居所地在境外。
  3. 单位是外国企业或者港澳台企业。
  4. 劳动合同履行地在港澳台地区、外国、公海、南极大陆等境外地区。

涉外劳动合同与境内劳动合同在部分规定上存在区别。

外国人

根据《外国人在中国就业管理规定》，外国人在中国内地工作，必须取得《外国人就业许可证》（就业证）和《外国人居留证》。其中，就业证由聘用的单位为其申请，外国人就业的单位必须与就业证上注明的单位一致。

持有《外国专家证》、《外国人在中华人民共和国从事海上石油作业工作准证》或者《临时营业演出许可证》的外国人，根据《外国人在中国就业管理规定》第九条的规定从事相关工作，无须办理就业证。

外国人仅在华就业的准入受到限制，一旦获得在华就业资格，则享受国民待遇，受中国《劳动法》保护。

《外国人在中国就业管理规定》【2017】 第十七条 用人单位与被聘用的外国人应依法订立劳动合同。劳动合同的期限最长不得超过五年。劳动合同期限届满即行终止，但按本规定第十九条的规定履行审批手续后可以续订。

根据上述规定，单位与外国人签订的劳动合同期限不得超过5年。若单位未办理就业证延期手续，劳动合同期满时，就业证即行失效。

提问：单位与外国人签订的劳动合同期限长于五年（包括无固定期限劳动合同），是否有效？

单位与外国籍劳动者签订劳动合同的期限长于五年（包括无固定期限劳动合同），该劳动合同仍然有效。在就业证期限到期前，单位应当为劳动者申请办理就业证延期手续。

单位在就业证到期前拒绝为劳动者办理就业证延期手续，导致劳动合同事实无法继续履行的，属于违法终止劳动合同，需要承担赔偿金。外国籍劳动者也可以以单位“未按照劳动合同约定提供劳动保护或者劳动条件”为由解除劳动合同，并主张补偿金。（2018）京02民终1661号

单位在就业证到期前为劳动者申请办理就业证延期手续，但未获主管部门批准，则应当以情势变更为由解除劳动合同，并支付补偿金。

港澳台居民

自2018年7月28日起，港澳台居民不再需要办理《台港澳人员就业证》。在内地就业时，只需要使用港澳台居民居住证、港澳居民来往内地通行证、台湾居民来往大陆通行证等身份证件即可。

外国企业

根据《劳动法》第二条和《劳动合同法》第二条，我国劳动法有关规定只适用于在中国境内的单位。在中国境内是指，注册地在中国或者主营业地在中国的单位。

根据《国务院关于管理外国企业常驻代表机构的暂行规定》，外国企业常驻代表机构聘请工作人员，应当通过涉外就业服务单位办理。直接招用劳动者的，不属于劳动关系，而属于雇佣关系。

外国企业或者其驻华代表机构直接招用劳动者，双方签订的劳动合同无效，不适用中国劳动合同法。（2016）京民申439号

境外劳务派遣

国内企业可以与外国企业签订劳务派遣协议，将国内劳动者向境外劳务派遣。经营境外劳务派遣的企业必须取得《对外劳务合作经营资格证书》和《境外就业经营许可证》，且为劳动者办理的是工作签证。

由于境外劳务派遣本质仍属于劳务派遣，因此劳动者与国内派遣单位之间仍然存在劳动关系。

注意，单位不得向境外劳务派遣劳动者收取履约保证金。

法律适用

涉外劳动合同与普通涉外合同不同，不允许在劳动合同中约定适用的法律。虽然不允许约定适用的法律，但并不代表必须适用中国法。

《涉外民事关系法律适用法》【2010】 第四十三条 劳动合同，适用劳动者工作地法律；难以确定劳动者工作地的，适用用人单位主营业地法律。劳务派遣，可以适用劳务派出地法律。
《最高人民法院关于适用〈中华人民共和国涉外民事关系法律适用法〉若干问题的解释（一）》【法释〔2012〕24号】 第十条 有下列情形之一，涉及中华人民共和国社会公共利益、当事人不能通过约定排除适用、无需通过冲突规范指引而直接适用于涉外民事关系的法律、行政法规的规定，人民法院应当认定为涉外民事关系法律适用法第四条规定的强制性规定： （一）涉及劳动者权益保护的； ...

据此，涉外劳动合同的法律适用按照下列规定确定：

  1. 涉及劳动者权益保护的，适用中国法。
  2. 劳动合同中其他不涉及劳动者权益保护的内容，适用劳动者工作地法律。难以确定工作地的，适用用人单位主营业地法律。
  3. 劳务派遣合同中不涉及劳动者权益保护的内容，由法院选择，可以适用劳动者工作地法律，工作地难以确认的，适用单位主营业地法律；也可以直接适用劳务派出地法律。

在司法实践中，由于对外国法律的适用比较麻烦。但凡允许适用中国法律的，法院一般会直接适用中国法律。
