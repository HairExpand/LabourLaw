第四十五章 劳动合同的解除（主动辞职）

《劳动合同法》【2012】 第三十七条 劳动者提前三十日以书面形式通知用人单位，可以解除劳动合同。劳动者在试用期内提前三日通知用人单位，可以解除劳动合同。

概述

主动辞职，是一种任意、劳动者单方、预告、无过错解除劳动合同的方式。

在劳动关系中，劳动者相对于用人单位而言始终处于弱势地位，从保护劳动者权益出发，法律赋予了劳动者单方任意解除劳动合同的权利。主动辞职的特点如下：

（1）主动辞职是一种任意解除劳动合同的方式。劳动者提出解除劳动合同时，无须向用人单位说明理由。劳动者既可以不说明辞职的理由就解除劳动合同，劳动者也可以以任何理由解除劳动合同。

（2）主动辞职是一种劳动者单方解除劳动合同的方式。劳动者提出解除劳动合同，无须经用人单位审批或者同意。用人单位不同意解除劳动合同，不影响预告期满后劳动合同发生解除的法律效力。

（3）主动辞职不受劳动合同期限的限制。无论劳动合同的期限有多长，甚至是无固定期限劳动合同，劳动者都有权单方解除劳动合同。需要注意的是，在法律规定的特殊情形下，劳动者主动辞职受到一定限制，如服务期或者脱密期。除此法律规定的情形外，劳动者主动辞职不受任何限制。

（4）主动辞职是一种预告解除劳动合同的方式。劳动者主动辞职时，必须履行预告程序。

预告程序

主动辞职的预告程序，是指劳动者主动辞职应当提前通知用人单位。

法律之所以规定劳动者主动辞职需要履行预告程序提前通知用人单位，是给双方足够的时间交接工作，便于用人单位及时安排人员接替，确保正常的工作秩序，避免因解除劳动合同影响单位的生产经营活动，给单位造成不必要的损失。在预告期限内，单位应当及时安排工作交接，劳动者需要充分配合。如果预告期限届满，工作仍然无法交接完毕，劳动者不需要承担责任。无论用人单位是否批准劳动者的主动辞职，法定预告期限届满，即发生解除劳动合同的法律效果。

主动辞职的预告程序有期限和形式两方面要求。

预告期限 预告形式
正式期 30日 书面
试用期 3日 书面/口头

预告期限

正式期劳动者解除劳动合同的，应当提前30日通知单位；试用期劳动者解除劳动合同的，应当提前3日通知单位。

注意“30日”和“3日”期间的计算规则：

（1）以自然日计算，法定休假日包含在内。

（2）预告期自劳动者提出解除劳动合同的次日起计算。

（3）最后一日是法定休假日的，应延期至法定休假日后的第一个工作日。

例如，正式期劳动者在2021年2月25日（周四）提出解除劳动合同，则期限的最后一日是2021年3月27日（周日）。因2021年3月27日是法定休假日，因此延后到2021年3月28日（周一）。劳动者自2021年3月29日起与用人单位之间不再存在劳动关系。

提问：劳动者提前30日通知单位解除劳动合同后，单位能否要求不满30日即解除劳动合同？

假如你提前30日通知单位即将辞职，单位领导却让你赶紧滚蛋，居然还要再等30天，一天都不想再看到你了。那么你当然可以立马走了，很多人辞职的时候都巴不得赶紧离开单位。但是注意这里的法律风险，必须取得离职证明后才能不去上班，否则单位可能以旷工或者未履行通知义务造成损失为由主张赔偿。

但是，有的劳动者可能已经作好安排，在30日后才能入职下一家单位，不愿意提前解除劳动合同，希望在满30日时才解除劳动合同。单位在劳动者提出辞职后，在30日的预告期限内反将劳动者辞退，或者要求提前解除劳动合同，是否合法？

司法实践中对此存在一定争议。

大部分法院认为，由于劳动者已经提出解除劳动合同，30日后才能解除是保护单位的合法权益，给单位足够的时间安排交接工作。之后因单位的行为（例如开除）提前达成了解除劳动合同的目的，这种情况仍然属于劳动者单方解除劳动合同，视为单位放弃30日的期限利益。因此，单位不需要承担补偿金或者赔偿金。（2017）粤01民终18980号

少部分法院认为，提前30日通知解除劳动合同是劳动者的权利，劳动合同自通知后满30日才能达到解除的效果。单位在30日内提前解除劳动合同的行为，不发生劳动合同解除的法律效力。（2016）沪02民终252号

预告形式

正式期劳动者解除劳动合同的，应当以书面形式通知用人单位；试用期劳动者解除劳动合同的，可以以口头形式通知用人单位。在劳务派遣中，被派遣劳动者通知的对象是派遣单位而非用工单位。

书面形式可以是辞职信或者辞职报告等形式，也可以根据用人单位规章制度在OA系统上提交电子申请，还可以发电子邮件。

试用期劳动者主动辞职既可以书面形式通知单位，也可以口头形式通知单位，但是，口头形式的通知难以固定为证据，如果觉得可能存在法律风险，建议还是以书面形式来主动辞职。

主动辞职的法律后果

劳动合同解除

劳动者以主动辞职的方式提出解除劳动合同，属于附期限的法律行为，提出时并不发生劳动合同解除的法律效果，劳动者合同仍然有效，双方应当继续履行劳动合同。预告期限内，劳动者应当提供劳动，单位应当支付劳动报酬。30日或者3日的预告期限届满，发生劳动合同解除的法律效力。

补偿金

劳动者主动辞职，单位无须向劳动者支付补偿金。实践中，单位想要开除劳动者时，往往采用“劝退”的方式要求劳动者主动解除劳动合同，诱使劳动者签订辞职申请书、离职意向书等等，从而达到劳动者主动辞职的法律效果。在这种情况下，劳动者应当坚决予以拒绝，不轻易签订任何文件。

竞业限制

如果存在竞业限制，劳动者主动辞职后，应当履行竞业限制义务。

服务期

劳动者在服务期内，不得以主动辞职的方式解除劳动合同，否则构成服务期违约，需要承担违约金。

违法主动辞职

预告程序是劳动者主动辞职必须履行的法律义务。劳动者没有提前通知单位，或者提前通知但未经单位同意在预告期限内不辞而别，构成违法解除劳动合同。根据《劳动合同法》【2012】第九十条，劳动者违反法律规定解除劳动合同，给用人单位造成损失的，应当承担赔偿责任。因此，劳动者千万不要在OA上提了离职申请，或者递交了辞职信后，就等不及立马走人，这种行为具有较大的法律风险，可能承担违约赔偿责任。所以，为了避免法律风险，最好保留已经提前书面通知单位解除劳动合同的证据，例如OA申请截图、辞职信签收回执、与主管的聊天记录等等。
